# 옛날 메이플 레벨업 계산기 & 타이머
빅뱅 이전 메이플스토리 경험치 테이블을 기반으로 제작된 레벨업 소요 시간 계산기입니다.

## 주요 기능
* 현재 레벨 및 목표 레벨 기반 필요 경험치 계산
* 현재 보유 경험치를 퍼센트(%)로 입력 가능
* 10분 사냥 효율 측정용 타이머 (종료 알림음 포함)
* cheer up button

## 실행 방법1
1. Python 설치
2. maple_calc.py 실행

## 실행 방법2
1. Release 기능에서 .exe파일 다운로드 후 windows 환경에서 실행.
